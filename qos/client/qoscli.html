# QOS Client

提供与QOS交互的命令行工具，主要提供以下命令：

  • keys 本地密钥库相关操作
  • query 信息查询操作
  • tx 交易执行操作
  • tendermint tendermint自带指令
  • version 版本信息

执行qoscli help可查阅qoscli包含的指令说明。

# 密钥（keys）

本地密钥库主要包含以下指令：

  • add 新增密钥
  • list 显示密钥列表
  • update 更新密钥保存密码
  • delete 从密钥库删除密钥
  • import 导入密钥
  • export 导出密钥

新增（add）

qoscli keys add <key_name> <key_name>可随意填写，仅作为本地密钥库密钥区分。

# 交易（tx）

执行

$ qoscli tx --help
1

可以看到QOS提供以下几种交易：

  • transfer 转账
  • create-approve 创建预授权
  • increase-approve 增加预授权
  • decrease-approve 减少预授权
  • use-approve 使用预授权
  • cancel-approve 取消预授权
  • create-qsc 创建联盟币
  • issue-qsc 发放联盟币
  • init-qcp 初始化联盟链

转账（transfer）

qoscli tx transfer --senders <senders_and_coins> --receivers <receivers_and_coins>

主要参数：

  • senders 发送集合，账户传keystore name 或 address，多个账户半角分号分隔
  • receivers 接收集合，账户传keystore name 或 address，多个账户半角分号分隔

Peter向地址address1t7eadnyl8g6ct9xyrasvz4rdztvkeqpc0hzujh转账1个qos，1个qstar

$ qoscli tx transfer --senders Peter,1qos,1qstar --receivers address1t7eadnyl8g6ct9xyrasvz4rdztvkeqpc0hzujh,1qos,1qstar
